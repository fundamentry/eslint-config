# `@fundamentry/eslint-config-typescript-react`

TypeScript React ESLint configuration.

## Installation

```sh
pnpm install -D @fundamentry/eslint-config-typescript-react
```

## Usage

Extend this configuration in your project's `.eslintrc.json`:

```json
{
  "$schema": "https://json.schemastore.org/eslintrc",
  "extends": "@fundamentry/eslint-config-typescript-react"
}
```
