# `@fundamentry/eslint-config-json`

JSON ESLint configuration.

## Installation

```sh
pnpm install -D @fundamentry/eslint-config-json
```

## Usage

Extend this configuration in your project's `.eslintrc.json`:

```json
{
  "$schema": "https://json.schemastore.org/eslintrc",
  "extends": "@fundamentry/eslint-config-json"
}
```
